# The-world-of-character
The-world-of-character是一个开放世界2D字符游戏，支持热更新。
# 兼容性
[Windows]支持ANSI转译序列的Windows(一般为Windows10及以上)，需要系统有curl  
[Linux]支持ANSI转译序列的终端，需要系统有curl  
[MacOS]目前不提供官方支持。
